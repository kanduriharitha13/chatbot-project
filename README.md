# chatbot-project
Interactive AI chatbot built with Node.js featuring voice support, chat history, songs (Taylor &amp; Selena), and subject explanations
# 🤖 AI Chatbot

A smart and interactive chatbot built using HTML, CSS, JavaScript, and Node.js.
This chatbot can handle casual conversations, explain subjects, and play songs from popular artists.

---

## ✨ Features

* 💬 **Casual Chatting** (100+ responses)
* 📚 **Subject Definitions**

  * Python 🐍
  * Java ☕
  * HTML 🌐
  * CSS 🎨
  * DBMS 🗄
  * CN 🌍
  * OS 💻
* 🎶 **Music Section**

  * Taylor Swift songs 🎤
  * Selena Gomez songs 🎤
* 🎤 **Voice Support**

  * Voice Input (Speech Recognition)
  * Voice Output (Text-to-Speech)
* 🌗 **Dark Mode / Light Mode**
* 🕓 **Chat History Saved (Local Storage)**
* 📂 **Sidebar History View**
* ⚡ **Typing Animation**
* 🎨 **Modern UI Design with Animations**

---

## 🚀 How to Run the Project

### 1️⃣ Install Node.js

Download and install Node.js

---

### 2️⃣ Open Project Folder

```bash
cd chatbot_project
```

---

### 3️⃣ Install Dependencies

```bash
npm install
```

---

### 4️⃣ Start Server

```bash
node server.js
```

---

### 5️⃣ Open in Browser

```
http://localhost:3000
```

---

## 🎯 How to Use

* Type **"subjects"** → get subject list
* Type **"python", "java", etc.** → get definitions
* Type **"songs"** → choose artist
* Type **"taylor" / "selena"** → play songs
* Use 🎤 for voice input
* Change theme using 🌙 / ☀️ button
