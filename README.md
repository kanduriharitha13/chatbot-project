# chatbot-project
Interactive AI chatbot built with Node.js featuring voice support, chat history, songs (Taylor &amp; Selena), and subject explanations
# 🤖 Smart AI Chatbot

An interactive AI chatbot built using **Node.js, HTML, CSS, and JavaScript**.

---

  # Features

- 💬 Chat with bot (text & voice)
- 🔊 Voice input & voice reply
- 🎶 Songs (Taylor Swift & Selena Gomez)
- 📚 Subject explanations (Python, Java, HTML, CSS, DBMS, CN, OS)
- 😊 100+ casual conversations
- 💾 Chat history sidebar
- ⌨️ Typing animation
- 🎨 Beautiful UI with background image

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js

### 1. Install dependencies

npm install


### 2. Start server

node server.js


### 3. Open in browser

http://localhost:3000
